# ReactConcepts

## How to install React?
`npx create-react-app my-app` 

Move into that file:
`cd my-react-app`

Start the terminal:
`npm start`


ts commans in the terminal
`npx tsc`

`tsc --init`
  Creates a tsconfig.json with the recommended settings in the working directory.

  download types:
  `npm i --save-dev @types/react`
